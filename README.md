# 📘 HSC Physics Important Topics AI Agent

A web-based AI study assistant for 12th HSC Maharashtra Board Physics students.

This project highlights important topics chapter-wise based on analysis of previous year question papers and official question banks.

## 🚀 Features

- Covers all 16 chapters of HSC Physics (Maharashtra Board)
- Highlights:
  - 🔥 High Priority Topics
  - ⭐ Medium Priority Topics
- Based on Previous Year Paper Trends
- Clean and Responsive UI
- Works Offline (No Backend Required)
- Lightweight and Fast

## 📚 Chapters Covered

1. Rotational Dynamics  
2. Mechanical Properties of Fluids  
3. Kinetic Theory of Gases & Radiation  
4. Thermodynamics  
5. Oscillations  
6. Superposition of Waves  
7. Wave Optics  
8. Electrostatics  
9. Current Electricity  
10. Magnetic Effects of Electric Current  
11. Magnetism  
12. Electromagnetic Induction  
13. Alternating Current  
14. Dual Nature of Radiation & Matter  
15. Atoms & Nuclei  
16. Semiconductor Devices  

## 🛠️ Technologies Used

- CSS3

- No backend
